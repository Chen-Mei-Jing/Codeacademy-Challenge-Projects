HTML Tags

Tags

Name

Description
<html > HTML element The <html> element should be added after the !DOCTYPE declaration. All content/structure for an HTML document should becontained between the opening and closing <html> tags.
<head> HTML Head Element The head element contains general information about an HTML page that isnâ€™t displayed on the page itself. Metadata includes things like the title of the HTML document and links to stylesheets.
<body> HTML Body Element The body element represents the content of an HTML document. Content inside <body> tags are rendered on the web browsers.
<p> Paragraph Element The paragraph element contains and displays a block of text.
<h1><h2>...<h6> Heading Elements HTML can use six different levels of heading elements ordered from the highest level <h1> to the lowest level <h6> .
<img> Image Element HTML image <img> elements embed images in documents. The src attribute contains the image URL and is mandatory.
<a> Anchor Element This element is used to create the hyperlinks that can point to other webpages, files on the same server, a location on the same page, or any other URL via the hyperlink reference attribute, <href>.

HTML Attributes

Attributes

Names

Description
src Src Attribute The src attribute contains the image URL and is mandatory. <img> is an empty element meaning it should not have a closing tag.
alt Alternative Attribute An <img> element can have alternative text which will be displayed if an image fails to render due to an incorrect URL, if the image is blocked from being displayed, or if the image has not been received from the URL.
href Hyperlink reference Attribute The href determines the location the anchor element points to.
target Target Attribute The target attribute on an <a> specifies where a hyperlink should be opened. A target value of "_blank" will tell the browser to open the hyperlink in a new tab in modern browsers if the browser has had settings changed to open hyperlinks in a new window.

HTML tables

Tags

Names

Description
<table> Table the <table> element has content that is used to represent a two-dimensional table made of rows and columns.
<thead> Table Head The table head element defines headings of table columns encapsulated in table rows or the set of rows defining the column headers in a table.
<tbody> Table Body The table body element contains all table row elements, and indicates that <tr> elements make up the body of the table. It is the set of rows containing actual table data.
<tr> Table Row The table row element is used to add rows to a table before adding table data and table headings.
<td> Table Data The table data element is nested inside a table row element <tr> to add a cell of data to a table.
<tfoot> Table Foot The table foot element uses table rows to give footer content or to summarize content at the end of a table.

CSS Properties

Property

Values

Description

font-size font-size : value px ; The font-size CSS property is used to set text sizes. Font size values can be many different units or types such as pixels.
font-family font-family : font name; The font-family CSS property is used to specify the typeface in a rule set. Fonts must be available to the browser to display correctly.
font-weight font-weight : value(bold or normal); The font-weight CSS property can be used to set the weight (boldness) of text.
text-align text-align : value(left, right, center) The text-align CSS property can be used to set the text alignment of inline contents.
color color : value( green or #22f) Using the color property, foreground text color of an element can be set in CSS which has the value of color like green or blue . Also, 3 digit or 6 digit color code like #22f or #2a2aff to set the color.
